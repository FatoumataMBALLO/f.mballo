### 📌 6. [Web Scraping & Analyse de données – IMDb (2022)](#-6-web-scraping--analyse-de-données--imdb-2022)

**Contexte**
Projet de web scraping et d’analyse de données visant à collecter et exploiter des informations sur les films sortis en 2022 à partir du site IMDb, classés par recettes au box-office américain.

**Objectifs**

* Automatiser la collecte de données web
* Structurer des données non structurées issues du HTML
* Préparer les données pour des analyses décisionnelles

**Démarche**

* Web scraping avec **Python**
* Extraction HTML via **BeautifulSoup**
* Structuration des données avec **Pandas**
* Nettoyage, typage et export des données

**Analyses réalisées**

* Classement des films par performance financière
* Analyse des notes IMDb et de la popularité
* Comparaison par genres
* Identification de tendances du marché cinématographique

**Compétences**
Python · Web Scraping · BeautifulSoup · Pandas · Data Cleaning · Data Analysis
